## Panoramica

MiroFish e un motore di previsione AI di nuova generazione basato su tecnologia multi-agente. Estrae semi informativi dal mondo reale (notizie, bozze di policy, segnali finanziari) e costruisce automaticamente un mondo digitale parallelo ad alta fedelta, in cui migliaia di agenti con personalita, memoria di lungo termine e logiche comportamentali interagiscono e si evolvono.

Con una prospettiva "dall'alto", puoi iniettare variabili e simulare traiettorie future per supportare decisioni migliori.

## Avvio Rapido

1. Configura variabili in `.env` a partire da `.env.example`.
2. Installa dipendenze:

```bash
npm run setup
npm run setup:backend
```

3. Avvia frontend + backend:

```bash
npm run dev
```

- Frontend: `http://localhost:3000`
- Backend API: `http://localhost:5001`
